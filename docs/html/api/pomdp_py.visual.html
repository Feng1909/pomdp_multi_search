pomdp_py.visual.search_tree module¶

pomdp_py.visual.search_tree.visualize_pouct_search_tree(root, max_depth=1, visit_threshold=1, anonymize=False, anonymize_actions=False, anonymize_observations=False, output_file=None, use_dot=False, ax=None)[source]¶

Visualize the given tree up to depth max_depth. Display nodes with number of visits >= visit_threshold.

Caveat: This only works well if the search tree depth is shallow. For larger trees, please use a combination of tree debugger (utils.debugging.TreeDebugger) and visualizer (custom in heritance of visual.visualizer.Visualizer).

If anonymize is True, will only display actions as a1,a2,… and observations as o1,o2,… .
